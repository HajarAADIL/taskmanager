# 📝 Task Manager API

Une API REST simple pour gérer des tâches, construite avec **Spring Boot** et une base de données **H2** en mémoire.

## 🚀 Fonctionnalités

- 🔍 Lister toutes les tâches
- ✅ Ajouter une nouvelle tâche
- ✏️ Modifier une tâche existante
- ❌ Supprimer une tâche
- 📄 Documentation Swagger (à venir)

## 🛠️ Technologies utilisées

- Java 17+
- Spring Boot 3.5
- Spring Data JPA
- H2 Database
- Maven
- Lombok
